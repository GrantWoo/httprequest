# httpurlconnectionwrapper
HttpUrlConnection wrapper for Android written in Java.

## Functionality
Provides wrappers for the following HTTP requests:

    HTTP GET
    HTTP POST
    HTTP PUT
    HTTP DELETE

With getters and setters of:

    Request headers
    Request body
    Response headers
    Response body (uncompressed, gzip)
    Request cookies
    Response cookies
    Connection timeout
    Read timeout
    Redirect URL
